### 功能说明

| 功能 | 说明 |
|------|------|
| **Correction** | 角度校准文件 (.dat/.csv 格式) |
| **Firetime** | 发光时间校准文件，用于运动补偿 |
| **光心修正** | 光学中心到几何中心的坐标变换 |

### 分帧方式说明

| 分帧方式 | 说明 |
|----------|------|
| **角度分帧** | 监测水平角度跳变，角度差值超过阈值时分帧（旋转式雷达） |
| **帧ID分帧** | 数据包中包含帧序号，序号变化时分帧（固态/半固态雷达） |
| **列ID分帧** | 基于列序号的回绕检测分帧（FT120 固态雷达） |
